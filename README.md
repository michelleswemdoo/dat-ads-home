# DatAds Take-Home

## What’s in the repo

- **`part_1/`** — System design and architecture (`part_1/README.md`, plus diagram path noted there).
- **`part_2/`** — Working TypeScript pipeline: mock APIs (Facebook, Google, TikTok), SQLite `insight` table, aggregated metrics (`part_2/README.md`).

Part 3 (query API server) is not implemented; query/caching/replica ideas live in the Part 1 write-up.

## Layout

```text
datads-take-home/
├── part_1/
├── part_2/
└── README.md
```

## Run Part 2

```bash
cd part_2
npm install
npm run start
```

Use **`part_2/README.md`** for Node version, `.env`, scripts (`start.sh`, etc.), and behavior details.
